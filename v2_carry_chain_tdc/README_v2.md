# 版本2: 进位链TDC设计

## 📋 文件说明

- **tdc_top.v** - 进位链TDC主模块
- **tb_tdc_top.v** - 进位链TDC测试平台
- **tdc_analysis.txt** - 性能分析报告

## ✅ 改进特性

1. **使用FPGA进位链** - 利用CARRY4原语实现精确延迟
2. **完全可综合** - 所有代码都可综合到实际硬件
3. **更高精度** - 分辨率提升到25ps
4. **更大量程** - 支持167ms测量范围
5. **完善同步** - 三级同步器防止亚稳态

## 🎯 适用场景

- 一般精度时间测量应用
- 需要平衡性能和复杂度的项目
- 学习和理解进位链TDC原理

## 📊 性能指标

- **分辨率**: 25ps (理论值)
- **量程**: 167ms (24位粗计数)
- **时钟频率**: 100MHz
- **可综合性**: ✅ 完全可综合
- **资源使用**: 优化 (使用专用进位链)
- **推荐度**: ⭐⭐⭐⭐ (推荐用于一般应用)

## 🚀 快速使用

```verilog
// 在你的设计中实例化
tdc_top #(
    .COARSE_WIDTH(24),
    .FINE_WIDTH(8)
) your_tdc (
    .clk(your_100mhz_clk),
    .rst_n(your_reset_n),

    .valid(result_valid)

```

## 🔧 使用建议

1. 使用100MHz时钟
2. 确保输入信号上升时间快
3. 参考tdc_analysis.txt了解详细设计
4. 运行tb_tdc_top.v验证功能
