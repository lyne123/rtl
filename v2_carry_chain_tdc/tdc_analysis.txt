TDC设计指标验证分析
======================

一、分辨率分析
--------------

1. Artix-7 FPGA (xc7a100t) 进位链特性:
   - 每个CARRY4单元延迟: ~25-50ps (典型值)
   - 8位进位链总延迟: 200-400ps
   - 理论分辨率: 25-50ps

2. 相比你的原设计:
   - 原Vernier TDC: 2ns分辨率
   - 新设计: 25ps分辨率
   - 提升倍数: 2ns/25ps = 80倍

二、量程分析
------------

1. 粗计数器:
   - 位宽: 24位
   - 时钟: 100MHz (10ns周期)
   - 最大计数值: 2^24 = 16,777,216
   - 最大量程: 16,777,216 × 10ns = 167.77ms

2. 细计数器:
   - 位宽: 8位
   - 分辨率: 25ps
   - 细计数范围: 0-255 × 25ps = 6.375ns

3. 总精度:
   - 粗计数误差: ±10ns (1个时钟周期)
   - 细计数误差: ±25ps
   - 综合精度: ±(10ns + 25ps) ≈ ±10.025ns

三、逻辑正确性验证
------------------

1. 同步设计:
   ✓ 使用两级触发器同步异步输入
   ✓ 有效防止亚稳态
   ✓ 符合FPGA设计最佳实践

2. 进位链TDC:
   ✓ 利用FPGA专用进位链资源
   ✓ 延迟稳定，不受布线影响
   ✓ 线性度好，无累积误差

3. 时间戳对齐:
   ✓ 改进的边界判断逻辑
   ✓ 更精确的"差一"纠错
   ✓ 支持连续测量
