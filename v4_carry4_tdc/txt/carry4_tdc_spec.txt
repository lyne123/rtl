1. 设计概述
================================================================================

1.1 项目名称
    基于CARRY4进位链的高精度时间数字转换器 (TDC)

1.2 设计版本
    v4_carry4_tdc

1.3 目标芯片
    Xilinx Artix-7 XC7A100TFGG484-2

1.4 设计目标
    - 测量精度: ~50ps (实际) / 42ps (理论)
    - 测量范围: 0 ~ 10.7秒
    - 输入信号: PWM波
    - 系统时钟: 400MHz (50MHz × 8)

2. 系统架构
================================================================================

2.1 整体架构
    ┌─────────────┐    ┌─────────────┐    ┌─────────────────┐
    │   50MHz     │───►│    MMCM     │───►│   400MHz        │
    │   晶振      │    │   ×8倍频    │    │   系统时钟      │
    └─────────────┘    └─────────────┘    └─────────┬───────┘
                                                    │
    ┌─────────────┐    ┌─────────────────────────────────────────┐
    │   PWM输入   │───►│        边沿检测与同步模块              │
    └─────────────┘    └───────────────────┬─────────────────────┘
                                           │
                   ┌───────────────────────┼─────────────────────┐
                   ▼                       ▼                     ▼
        ┌─────────────────────┐ ┌─────────────────┐ ┌─────────────────┐
        │   粗计数器模块      │ │ CARRY4延迟链    │ │ 时间戳合成器   │
        │   (400MHz, 32位)   │ │ (60级, 6位)     │ │ (38位输出)      │
        └─────────────────────┘ └─────────────────┘ └─────────────────┘
                   │                       │                     │
                   └───────────┬───────────┴─────────┐           │
                               ▼                     ▼           ▼
                       ┌─────────────────────────────────────────┐
                       │         温度计码解码器                  │
                       │         (60位→6位)                      │
                       └─────────────────────────────────────────┘

2.2 模块功能

    2.2.1 MMCM倍频模块 (mmcm_50m_to_400m.v)
        - 输入: 50MHz参考时钟
        - 输出: 400MHz系统时钟
        - 倍频系数: ×8
        - 时钟质量: 偏斜<50ps, 抖动<20ps

    2.2.2 边沿检测与同步模块 (edge_detector_sync.v)
        - 功能: PWM信号同步和边沿检测
        - 同步级数: 三级同步器
        - 输出: start_edge, stop_edge, start_sync, stop_sync

    2.2.3 粗计数器模块 (coarse_counter_400m.v)
        - 时钟: 400MHz (2.5ns周期)
        - 计数位宽: 32位
        - 最大测量时间: 10.7秒

    2.2.4 CARRY4延迟链模块 (carry4_delay_chain.v)
        - 延迟级数: 60级CARRY4
        - 每级延迟: ~35ps
        - 总延迟: ~2.1ns
        - 输出: 60位温度计码

    2.2.5 温度计码解码器 (thermometer_decoder.v)
        - 输入: 60位温度计码
        - 输出: 6位二进制码
        - 算法: 并行计数

    2.2.6 时间戳合成器 (timestamp_synthesizer.v)
        - 输入: 粗计数值 + 细计数值
        - 输出: 38位时间间隔
        - 算法: 定点数计算

9. 应用指南
================================================================================

9.1 硬件连接

    - 50MHz晶振连接到clk_50m引脚
    - PWM信号源连接到pwm_in引脚
    - 复位信号连接到rst_n引脚
    - 测量结果通过time_interval输出

9.2 软件接口

    - 监控valid信号确定数据有效性
    - 读取time_interval获取测量结果
    - 时间单位换算: 实际时间 = time_interval × 41.7ps

9.3 校准方法

    - 使用标准时间信号源校准
    - 记录不同温度下的测量偏差
    - 建立校准系数表
    - 实时应用校准补偿

10. 设计优势
================================================================================

10.1 相比传统设计

    - 更高精度: CARRY4延迟比LUT更稳定
    - 更好线性度: 进位链延迟一致性更好
    - 更低温度漂移: ±5% vs ±10%
    - 硬件优化: 充分利用FPGA原语

10.2 相比v3版本

    - 精度提升: 从100ps提升到50ps
    - 稳定性提升: 温度漂移降低
    - 资源优化: 更高效的实现
    - 时序优化: 更好的时序约束

11. 设计限制
================================================================================

11.1 精度限制

    - CARRY4延迟受温度影响
    - 时钟抖动限制最终精度
    - 工艺偏差影响延迟一致性

11.2 应用限制

    - 输入信号需要足够的建立保持时间
    - 测量范围受粗计数器位宽限制
    - 需要稳定的时钟源
