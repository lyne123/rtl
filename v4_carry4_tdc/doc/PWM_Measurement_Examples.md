# 双TDC PWM信号测量示例文档

## 系统参数
- **粗计数时钟**：400MHz (2.5ns周期)
- **细计数精度**：34.7ps (理论)
- **测量范围**：35ps ~ 10.74秒
- **双TDC架构**：分别测量上升沿(a)和下降沿(b)

## 核心公式

### 标准计算公式
```
脉宽 = (N + 1) × 2.5ns - a - b
```
其中：
- N = 粗计数差值（stop_coarse_ts - start_coarse_ts）
- a = TDC-A测量值（PWM上升沿偏移）
- b = TDC-B测量值（PWM下降沿提前）

### 边界情况公式
```
当 perfect_alignment_detected = true 时：
脉宽 = N × 2.5ns
```

### 参数转换
```
a值（皮秒）= fine_a × 35ps
b值（皮秒）= fine_b × 35ps
粗计数时间 = N × 2500ps
```

## 可测量的PWM信号类型

### 类型1：极短脉冲（< 2.5ns）

#### 示例1.1：100ps脉宽
```
PWM参数：频率=10MHz，占空比=0.001%

假设时序：
- 400MHz时钟边沿：t=1.2ns
- PWM上升沿：t=1.2ns（a=1.2ns）
- PWM下降沿：t=1.3ns（b=1.2ns到下一个时钟）

测量结果：
- TDC-A测得：a = 1.2ns → fine_a = 34（1.2ns ÷ 35ps）
- TDC-B测得：b = 1.2ns → fine_b = 34（1.2ns ÷ 35ps）
- 粗计数：N = 0（同一时钟周期内）

公式计算：
脉宽 = (N + 1) × 2.5ns - a - b
     = (0 + 1) × 2.5ns - 1.2ns - 1.2ns
     = 2.5ns - 2.4ns
     = 0.1ns = 100ps ✅

验证：计算结果100ps = 实际脉宽100ps
应用场景：激光脉冲测量、高速信号检测
```

#### 示例1.2：1.8ns脉宽
```
PWM参数：频率=100MHz，占空比=18%

假设时序：
- 400MHz时钟边沿：t=0ns
- PWM上升沿：t=0.3ns（a=0.3ns）
- PWM下降沿：t=2.1ns（b=0.4ns到下一个时钟）

测量结果：
- TDC-A测得：a = 0.3ns → fine_a = 9（0.3ns ÷ 35ps）
- TDC-B测得：b = 0.4ns → fine_b = 11（0.4ns ÷ 35ps）
- 粗计数：N = 0

公式计算：
脉宽 = (N + 1) × 2.5ns - a - b
     = (0 + 1) × 2.5ns - 0.3ns - 0.4ns
     = 2.5ns - 0.7ns
     = 1.8ns ✅

验证：计算结果1.8ns = 实际脉宽1.8ns
应用场景：时钟信号分析、高速通信
```

### 类型2：短脉冲（2.5ns ~ 100ns）

#### 示例2.1：2.5ns脉宽（临界值）
```
PWM参数：频率=200MHz，占空比=50%

假设时序（完全对齐情况）：
- 400MHz时钟#1：t=0ns
- PWM上升沿：t=0ns（与时钟#1重合）
- 400MHz时钟#2：t=2.5ns
- PWM下降沿：t=2.5ns（与时钟#2重合）

测量结果：
- TDC-A测得：a = 0ns → fine_a = 0
- TDC-B测得：b = 0ns → fine_b = 0
- 粗计数：N = 1（跨越一个时钟周期）

边界检测：
perfect_alignment_detected = (fine_a ≤ 2) && (fine_b ≤ 2) = true

公式计算（对齐模式）：
脉宽 = N × 2.5ns
     = 1 × 2.5ns
     = 2.5ns ✅

验证：计算结果2.5ns = 实际脉宽2.5ns
应用场景：时钟信号、基准测试
```

#### 示例2.2：10ns脉宽
```
PWM参数：频率=50MHz，占空比=50%

假设时序：
- 400MHz时钟#1：t=0ns
- PWM上升沿：t=0.3ns（a=0.3ns）
- 400MHz时钟#5：t=10ns
- PWM下降沿：t=9.7ns（b=0.3ns到时钟#5）

测量结果：
- TDC-A测得：a = 0.3ns → fine_a = 9
- TDC-B测得：b = 0.3ns → fine_b = 9
- 粗计数：N = 3（跨越3个完整周期）

公式计算：
脉宽 = (N + 1) × 2.5ns - a - b
     = (3 + 1) × 2.5ns - 0.3ns - 0.3ns
     = 4 × 2.5ns - 0.6ns
     = 10ns ✅

验证：计算结果10ns = 实际脉宽10ns
应用场景：电机控制、LED调光
```

#### 示例2.3：50ns脉宽
```
PWM参数：频率=10MHz，占空比=50%

假设时序：

验证：计算结果50ns = 实际脉宽50ns
```

### 类型3：中等脉冲（100ns ~ 1μs）

#### 示例3.1：100ns脉宽
```
PWM参数：频率=5MHz，占空比=50%

假设时序：
- 400MHz时钟#1：t=0ns
- PWM上升沿：t=0.1ns（a=0.1ns）
- 400MHz时钟#41：t=100ns
- PWM下降沿：t=99.9ns（b=0.1ns到时钟#41）

测量结果：
- TDC-A测得：a = 0.1ns → fine_a = 3（0.1ns ÷ 35ps）
- TDC-B测得：b = 0.1ns → fine_b = 3
- 粗计数：N = 39

公式计算：
脉宽 = (N + 1) × 2.5ns - a - b
     = (39 + 1) × 2.5ns - 0.1ns - 0.1ns
     = 40 × 2.5ns - 0.2ns
     = 100ns ✅

相对误差分析：
绝对误差：±35ps
相对误差：±35ps ÷ 100ns = ±0.035%
应用场景：工业控制、传感器接口
```

```
PWM参数：频率=1MHz，占空比=50%

测量结果：

- TDC-B测得：b = 0.15ns → fine_b = 4

精度分析：
```

### 类型5：特殊波形

```
假设时序：
- 400MHz时钟：t=0ns
- PWM上升沿：t=0.5ns（a=0.5ns）

- TDC-B测得：b = 0.5ns → fine_b = 14

公式计算：
脉宽 = (0 + 1) × 2.5ns - 0.5ns - 0.5ns

双TDC优势：

```

### 类型6：边界情况

#### 示例6.1：时钟对齐PWM（改进后）
```
完全对齐情况：
- PWM上升沿：t=0ns（与时钟重合）
- PWM下降沿：t=2.5ns（与时钟重合）

测量结果：
- TDC-A测得：a = 0ns → fine_a = 0
- TDC-B测得：b = 0ns → fine_b = 0
- 粗计数：N = 1

改进前计算：
脉宽 = (1 + 1) × 2.5ns - 0 - 0 = 5.0ns ❌

改进后检测：
perfect_alignment_detected = (0 ≤ 2) && (0 ≤ 2) = true

改进后计算（对齐模式）：
脉宽 = N × 2.5ns = 1 × 2.5ns = 2.5ns ✅

改进效果：
- 从5.0ns错误结果修正为2.5ns正确结果
- 相对误差从100%降低到0%
```

## 精度分析表

| 脉宽范围 | 绝对精度 | 相对精度 | 主要误差源 |
|---------|---------|---------|-----------|
| < 2.5ns | ±35ps | 1-10% | TDC量化误差 |
| 2.5ns-100ns | ±35ps | 0.01-1% | TDC+时钟抖动 |
| 100ns-1μs | ±35ps | 0.001-0.01% | 时钟稳定性 |
| > 1μs | ±35ps | < 0.001% | 时钟长期稳定性 |

## 系统限制

**物理限制**：
- 最小脉宽：≈35ps（CARRY4延迟限制）
- 最大脉宽：≈10.74秒（32位计数器溢出）
- 输入频率：理论上无限制（异步采样）

**精度限制**：
- 理论精度：34.7ps
- 实际精度：50-100ps（包含时钟抖动等因素）
- 温度影响：延迟随温度变化
