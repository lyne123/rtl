# 版本1: 原始游标TDC设计

## 📋 文件说明

- **original_vernier_tdc.v** - 原始游标TDC核心模块
- **original_tdc_alignment.v** - 原始时间对齐核心模块

## ⚠️ 已知问题

1. **延迟单元不可综合**
   ```verilog
   assign #DELAY_VAL out = in; // 这行在综合时会被忽略
   ```

2. **精度有限** - 理论分辨率2ns
3. **延迟链级数少** - 只有5级
4. **MSB纠错逻辑简单** - 可能有边界问题

## 🎯 适用场景

- 学习游标TDC基本原理
- 理解时间数字转换概念
- 参考设计思路

## 📊 性能指标

- **分辨率**: 2ns
- **量程**: 有限 (取决于粗计数位宽)
- **可综合性**: ❌ 延迟单元问题
- **推荐度**: ⭐⭐ (仅作学习参考)
