# 时间戳计数法PWM脉宽测量模块

## 概述

本模块实现了一个基于时间戳计数法（Timestamping）的PWM脉宽测量系统，使用FPGA进行高精度时间测量。该系统能够在400MHz时钟下实现2.5ns的时间分辨率，支持0-50ns的脉宽测量范围。

## 架构设计

### 核心模块

1. **coarse_timestamp_generator.v** - 粗时间戳生成器
   - 400MHz全局自由运行计数器
   - 异步PWM信号同步和边沿检测
   - 时间戳捕获和状态机控制
   - 错误检测和溢出处理

2. **tdc_coarse_cnt_top.v** - 顶层集成模块
   - 集成粗时间戳生成器
   - 跨时钟域同步（400MHz → 50MHz）
   - 脉宽计算和结果输出
   - 完整的系统接口

3. **tb_coarse_cnt.v** - 测试平台
   - 完整的仿真验证环境
   - 多种测试场景覆盖
   - 精度验证和性能分析

## 关键技术特性

### 1. 时间戳计数法
- **全局计数器**：32位自由运行计数器，400MHz时钟驱动
- **边沿检测**：精确捕获PWM信号的上升沿和下降沿
- **时间戳记录**：分别记录T_rise和T_fall
- **脉宽计算**：Pulse_Width = T_fall - T_rise

### 2. 跨时钟域处理
- **异步信号同步**：3级同步器减少亚稳态
- **数据同步**：握手协议确保数据完整性
- **时钟域转换**：400MHz → 50MHz安全传输

### 3. 溢出处理
- **计数器溢出检测**：自动检测32位计数器回绕
- **时间戳计算**：正确处理溢出情况下的脉宽计算
- **错误指示**：提供溢出和异常状态指示

### 4. 错误检测机制
- **亚稳态防护**：多级同步器设计
- **异常边沿检测**：防止连续上升沿等异常情况
- **超时检测**：防止测量过程死锁
- **范围验证**：脉宽超出预期范围时报警

## 接口定义

### 顶层模块接口

```verilog
module tdc_coarse_cnt_top (
    input wire clk_50m,             // 50MHz系统时钟
    input wire clk_400m,            // 400MHz采样时钟
    input wire rst_n,               // 异步复位(低电平有效)
    input wire pwm_in,              // PWM信号输入
    output wire [37:0] time_interval, // 时间间隔输出(皮秒)
    output wire valid,              // 测量有效信号
    output wire measurement_error   // 测量错误指示
);
```

### 时序说明

1. **pwm_in**：异步PWM输入信号，相位与clk_400m完全不对齐
2. **time_interval**：测量结果，单位为皮秒(ps)
3. **valid**：测量结果有效指示，高电平表示新数据可用
4. **measurement_error**：错误指示，高电平表示测量异常

## 测量原理

### 1. 时间戳记录过程

```
全局计数器: 0    1    2    3    4    5    6    7    8
           |----|----|----|----|----|----|----|----|----|
           0ns  2.5ns 5ns  7.5ns 10ns 12.5ns15ns 17.5ns20ns

PWM信号:    _____/\______________________________/\______
           上升沿                          下降沿
           T_rise = 2                      T_fall = 6

脉宽计算: Pulse_Width = (6 - 2) × 2.5ns = 10ns
```

### 2. 溢出处理

```
当计数器达到最大值后回绕:

全局计数器: ... 4,294,967,294 4,294,967,295 0 1 2 3 ...
           |----|----|----|----|----|----|----|----|

PWM信号:    ________________________/\________________
           T_rise = 4,294,967,295    T_fall = 3

脉宽计算: Pulse_Width = (MAX_VALUE - T_rise) + T_fall + 1
```

## 使用说明

### 1. 集成到现有系统

```verilog
// 实例化TDC模块
tdc_coarse_cnt_top u_tdc (
    .clk_50m(sys_clk_50m),
    .clk_400m(clk_400m_from_mmcm),
    .rst_n(system_reset_n),
    .pwm_in(pwm_input_signal),
    .time_interval(measured_width),
    .valid(measurement_valid),
    .measurement_error(error_flag)
);
```

### 2. 时钟要求
- **clk_50m**：50MHz系统时钟
- **clk_400m**：400MHz采样时钟（可通过MMCM从50MHz生成）

### 3. 复位要求
- 异步复位信号，低电平有效
- 复位期间所有输出清零
- 复位释放后需要等待系统稳定

## 注意事项

1. **时钟质量**：确保400MHz时钟的抖动和偏斜在可接受范围内
2. **信号完整性**：PWM输入信号需要良好的信号质量
3. **复位时序**：复位释放后等待足够时间让系统稳定
4. **错误处理**：监控measurement_error信号，及时处理异常
5. **资源使用**：根据目标FPGA器件评估资源使用情况
